🔌 Kết nối Google AppScript
AppScript là backend lưu dữ liệu vào Google Sheets của bạn.
Dán URL /exec sau khi Deploy.
Hướng dẫn lấy URL
1. Mở Google Sheets → Extensions → Apps Script
2. Paste code từ blueprint → Save
3. Click Deploy → New deployment → Web app
4. Execute as: Me · Who has access: Anyone
5. Copy URL dạng /exec → dán vào ô trên
